Sending a message:
    Messages can be any length!
    To broadcast a message as a client, just write your message on the command line and hit enter
    To unicast a message to another client, put their name in square brackets before your message. E.g. [OtherUser] Hi there, other user!

Quitting:
    A user can disconnect from the server by entering /quit

Downloading files:
    A user can see the files available for download by typing /downloadList
    A user can download a file by typing /download [filePath] [fileName] where filePath is the location on the user's device where they want the file to go
    For example: /download ./ mobyDick.txt
